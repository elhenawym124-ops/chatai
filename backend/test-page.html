🧪 Facebook Test Server

اختبار الـ webhooks وإرسال الرسائل

📋 معلومات السرفر:

Server URL: http://localhost:3001
Webhook URL: http://localhost:3001/webhook
Health Check: http://localhost:3001/health

🔗 إعداد الـ Webhook في Facebook:

1. اذهب إلى Facebook Developer Console

2. اختر تطبيقك → Messenger → Settings

3. في قسم Webhooks، أضف:

Callback URL: https://your-ngrok-url.ngrok.io/webhook
Verify Token: test_webhook_token_123

4. اشترك في الأحداث: messages, messaging_reads, messaging_deliveries

📤 اختبار إرسال رسالة:

يمكنك الحصول عليه من Facebook Developer Console
ستحصل عليه من الرسائل الواردة في الـ logs
